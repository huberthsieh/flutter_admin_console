# 快速啟動指南

## 立即運行

### 1. 安裝依賴（如果還沒執行）
```bash
flutter pub get
```

### 2. 啟動開發伺服器
```bash
flutter run -d chrome
```

註：Flutter 3.38+ 版本會自動使用 CanvasKit 渲染器（已在 `web/index.html` 中配置）

## 使用說明

### 登入系統
1. 啟動應用程式後會自動打開登入頁面
2. 輸入任意帳號和密碼（目前為示範模式，只要不為空即可）
3. 點擊「登入」按鈕

### 瀏覽後台
登入後您可以：
- 查看儀表板統計資訊
- 點擊左側選單切換頁面
- 使用頂部搜尋功能
- 查看通知
- 點擊右上角用戶頭像進行設定或登出

### 響應式測試
- 調整瀏覽器視窗大小查看響應式效果
- 小於 768px 時側邊欄會自動收合
- 點擊左上角選單圖示可開啟側邊欄

```

  │
```

## 可用的頁面路由

- `/login` - 登入頁面
- `/` 或 `/home` - 首頁儀表板
- `/users` - 用戶管理
- `/settings` - 系統設定

## 常見問題

**Q: 如何登出？**
A: 點擊右上角用戶頭像，選擇「登出」

**Q: 為什麼頁面載入較慢？**
A: CanvasKit 首次需要下載約 2-3MB 的檔案，後續會被快取

**Q: 如何切換到 HTML 渲染器？**
A: 執行 `flutter run -d chrome --web-renderer html`

**Q: 支援手機瀏覽嗎？**
A: 支援！介面已做響應式設計，可在手機瀏覽器中正常使用
